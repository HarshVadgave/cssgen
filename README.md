# CSSGen

CSSGen is a free modern CSS UI generator that helps developers create beautiful Glassmorphism and Neumorphism components with live preview and copy-ready CSS.

## Features

- Glassmorphism CSS generator
- Neumorphism CSS generator
- Live component preview
- Copy CSS instantly
- Random style generator
- Export design as HTML + CSS
- Light and dark theme
- Responsive design
- FAQ section with SEO-friendly content

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Choose Glassmorphism or Neumorphism.
2. Adjust colors, blur, radius, opacity, shadow, width, and height.
3. Copy the generated CSS or export it as an HTML file.

## Project Description

CSSGen makes modern UI styling faster by letting users visually generate CSS effects instead of writing everything manually. It is useful for developers, students, and designers who want quick, clean, and customizable CSS components.
